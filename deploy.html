── DEPLOYMENT STEPS ──
STEP 01
GitHub Repo හදන්න
github.com/new ලට ගිහිල්ලා new repository එකක් හදන්න. ඊට පස්සේ source code eka upload කරන්න.
git init
git add .
git commit -m "first commit"
git push origin main
REQUIRED
STEP 02
MongoDB Atlas Setup
cloud.mongodb.com ලට ගිහිල්ලා free cluster එකක් හදන්න. Database user හදලා connection string eka copy කරගන්න.
mongodb+srv://user:pass@cluster.mongodb.net/botdb
REQUIRED
STEP 03
Render ලට Deploy
render.com ලට ගිහිල්ලා sign up කරන්න. "New Web Service" click කරලා GitHub repo eka connect කරන්න. Free plan select කරන්න.
FREE
STEP 04
Environment Variables
Render dashboard → Environment tab ලට ගිහිල්ලා MONGODB_URI variable eka add කරන්න. Build & start commands set කරන්න.
Build: npm install
Start: node pair.js
IMPORTANT
STEP 05
Bot Pair කරන්න
Deploy වෙලා ඉවර වුණාම Render URL eka copy කරගන්න. your-app.onrender.com ලට visit කරලා WhatsApp number eka enter කරලා pairing code eka ගන්න.
FINAL STEP
STEP 06
WhatsApp Connect
WhatsApp → Settings → Linked Devices → Link a Device → Link with Phone Number ලට ගිහිල්ලා pairing code eka ගහන්න. Bot connected! 🎉
